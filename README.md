# SoundBoard — Interactive Sound Pad

An interactive drum machine and sound pad with 4 sound packs, keyboard mapping, a 16-step beat sequencer, and loop recording — all synthesized in the browser with the Web Audio API.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
![Web Audio API](https://img.shields.io/badge/Web_Audio_API-4285F4?style=flat)

## Features

- **16-Pad Grid** — Tap or click pads to trigger sounds
- **Keyboard Mapping** — Play with Q-V, A-F, Z-L keys
- **4 Sound Packs** — Drums, Synth, FX, Lo-Fi
- **Beat Sequencer** — 16-step sequencer with adjustable BPM
- **Record & Playback** — Record your live performance and replay it
- **Visual Ripple Effects** — Animated feedback on every pad hit
- **BPM Control** — Adjust tempo from 40-300 BPM
- **Master Volume** — Global volume slider
- **No Audio Files** — All sounds synthesized in real-time via Web Audio API

## Getting Started

```bash
git clone https://github.com/TamerAli-0/sound-board.git
cd sound-board
# Open index.html in your browser
```

No dependencies, no build tools, no audio files needed.
